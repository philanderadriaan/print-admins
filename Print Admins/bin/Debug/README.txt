This program gives all printers in the server full permissions to the "NKSD PrintAdmins" group.

Instructions:
1. Extract the zip file and put "Print Admins.exe" and "SetACL.exe" in the server.
2. Right click "Print Admins.exe".
3. Left click "Run as administrator".
4. Allow the program to run if UAC pops up.
5. A console window will appear when running.
6. The program is done when the console window disappears.
